Meus Projetos

Este projeto, inspirado no Curso.dev, tem como objetivo recriar as funcionalidades do site tabnews.com.br, como páginas dinâmicas para publicação de conteúdo, autenticação de usuários, gerenciamento de permissões e interações em tempo real. Ele utiliza práticas modernas de desenvolvimento e ferramentas para garantir qualidade e consistência no código. Atualmente, o projeto está em desenvolvimento e conta apenas com a interface inicial que exibe esta mensagem. Também já está vinculado ao meu domínio próprio, prowulf.com.br, permitindo acompanhar o progresso e visualizar as atualizações futuras.

Este projeto, criado durante a Imersão Dev Back-End da Alura, teve como foco o desenvolvimento de um servidor funcional, com implementação de rotas, middleware e interação com banco de dados. Além das funcionalidades básicas, o projeto incorpora a integração com a IA Gemini, que permite analisar imagens e gerar descrições detalhadas com base em seu conteúdo, ampliando as possibilidades de uso e interação. Todas as funcionalidades foram testadas com Postman, seguindo boas práticas, como a configuração de variáveis de ambiente e o uso de uma arquitetura limpa e escalável.
